This is a modified task from freeCodeCamp.

The function converts the given number into a roman numeral once the button "convert" is clicked.

All roman numerals answers will be provided in upper-case.

There is also a function which check the input. In case the input is not a positive integer number, the alert "Only positive integer, please" is displayed.
